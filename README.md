# A-simple-calculator
This will just add, subtract,multiply and divide numbers entered but this is solely based on javascript
It accepts inputs (strictly just two inputs) , checks if they are numbers before performing the operations based on what you type in; thus, Add, Subtract, Multiply, or Divide. Note that, i stated in the code that the operations are case sensitive, so you type what you see.

I have also added a python version that runs on the terminal. The code is in two parts; the first part already has the numbers and so automatically does the calculations but the second part recieves the numbers as inputs from the user before the calculations are processed.
